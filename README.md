# SSNICIDE Code Editor 🚀
**By IncognitoM**

A high-performance, mobile-responsive web-based code editor designed for students and developers. It supports live HTML preview and real-time execution for C and Python.

## ✨ Features
* **Multi-Language Support:** Write and test HTML, C, and Python 3.
* **Live Preview:** Instant HTML rendering with auto-closing tags.
* **Textbook Compatibility:** Specifically optimized to handle Bangladesh textbook-style C code (void main, conio.h).[cite: 1]
* **Mobile Optimized:** Fully responsive UI for coding on Android and other mobile devices.[cite: 1]
* **Professional Editor:** Powered by the Monaco Editor (the core of VS Code).[cite: 1]

## 🛠️ Built With
* **HTML5/CSS3** (Custom UI/UX)[cite: 1]
* **Monaco Editor** for professional syntax highlighting.[cite: 1]
* **Piston API** for remote code execution.[cite: 1]
